Overview#

Dashboard is a web-based HKube user interface.

Dashboard is an implementation of Hkube Restful API.

Check Full Rest API documentation, and Swagger-UI.

Use Dashboard for deploying and monitoring user algorithms and pipelines.

Accessing the Dashboard UI#

  • https://<KUBERNETES-MASTER-IP>/hkube/simulator/

Features#

The Dashboard follows two main concepts Operations and Tables.

Operations#

Actions which deploying data to HKube. Use Operations sidebar for adding pipelines, algorithms etc.

Tables#

View Tables for monitoring HKube data. Navigate Tables panel for viewing information about pipelines structures, algorithm info etc.
